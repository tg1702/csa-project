Quine-McCluskey Algorithm

Prerequisites

You should be familiar with:

  • Writing Sum-of-Product expressions
  • Simplification using Boolean Algebra and Karnaugh maps

Objectives

On completion of this section, you should be able to:

  • Describe the Quine-McCluskey Algorithm
  • Know when to use it instead of other simplification techniques
  • Successfully perform the Quine-McCluskey Algorithm on Boolean expressions

Weâ€™ve already discussed two methods of minimizing Boolean expressions: simplification using Boolean algebra and simplification using Karnaugh maps. Recall from those notes that Karnaugh maps are the best choice when 4 or less variables are used in the expression. For situations where we have more than 4 variables, Quine-McCluskey is often the better choice. If you want to code a program that minimizes Boolean expressions, using the Quine-McCluskey algorithm in the program ensures that it can systematically calculate the expressions for 4, 10 and 20 variables.
