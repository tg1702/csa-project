S-R Flip-Flops

Objectives

On completion of this section, you should be able to:

  • What a S-R flip-flop is
  • How the S-R flip-flop operates
  • The characteristic table for a S-R flip-flop
  • Uses of S-R flip-flops

An S-R flip-flop boasts two inputs: Set (S) and Reset (R). When a signal is applied to the Set input, the output Qn+1 is "set" to 1. Conversely, when a signal is applied to the Reset input, the output Qn+1 changes to 0.

The S-R flip-flop can enter an illegal state known as a "race condition" where both the Set and Reset inputs are active at the same time, leading to unpredictable behavior. Nonetheless, S-R flip-flops are still widely used in digital circuits for tasks such as data storage, state machines, and synchronization.
