D Flip-Flops

Objectives

On completion of this section, you should learn:

  • What a D flip-flop is
  • The operation of a D flip-flop
  • Uses of D flip-flops
  • How to implement a D flip-flop using NAND and NOR gates

A D flip-flop is a type of digital flip-flop that has a single input called the D (or data) input. When the D input is 0, the output Q t+1 will be 0, and when the D input is set to 1 the output Q t+1 will be 1. The output will only reflect changes made to the D input if the clock signal is 1, otherwise the output will stay at 0.

In terms of their uses, D flip flops are used commonly in registers and counters, and they can be combined with other flip-flops to make more complex circuits, such as a master-slave flip-flop, which eliminates potential timing issues that arise in circuits with multiple flip-flops.
