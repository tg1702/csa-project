Using D Flip-flops to Build Bidirectional Synchronous Counters

Objectives

On completion of this section, you should be able to:

  • Understand the function of a bidirectional synchronous counter
  • Use D Flip-flops to design a bidirectional synchronous counter

What's a Bidirectional Synchronous Counter?

Simply put, a synchronous counter uses a clock pulse to trigger each flip-flop used in the counter at the same time. A bidirectional counter is a synchronous up/down binary counter that has the ability to count in both directions either to or from some preset value as well as zero. We'll use the 4-step process to help us in desgining our counter.

If you're unfamiliar with D flip-flops, take a look here then come back to this page: What are D flip flops?

The 4-Step Process

Although bidirectional synchronous counters take a bit more work, we can use the 4-step process from the section that dealt with designing synchronous counters. Let's look at the following example:

However, we need more information when working with bidirectional counters. As the word 'bidirectional' suggests, we're moving in two directions. We consider the directions for the 'up' count (M=0) and the 'down' count (M=1).

As you can see, the 'up' count goes in a clockwise direction (0,1,2,3,...) while the 'down' count moves counter-clockwise (0,7,6,...). This is the actual state diagram of this sequence.
